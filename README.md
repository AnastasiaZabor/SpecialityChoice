## Алгоритм программы

1. Начало алгоритма
2. Ввод размерности массива
3. Создание массива
4. Заполнение массива
    * Создание пустой строки 
    * Заполняем пустую строку случайными латинскими символами
5. Выводим массив
6. Делаем выборку нужных элементов из массива
7. Выводим результат
8. Конец алгоритма
